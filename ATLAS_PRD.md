1. WHAT IS ATLAS?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Atlas is a platform for organizations that manage multiple business
websites — clinic chains, restaurant franchises, real estate offices —
where every location needs its own site but everything must stay
on-brand and get approved before going live.

Think of it as the gap between Wix (too simple, zero control) and
Adobe AEM (too complex, $100K+/year). Atlas is powerful enough for
a corporate marketing team, simple enough for a front-desk receptionist.

THREE THINGS ATLAS DOES BETTER THAN ANYTHING ELSE:

  1. Build sites fast — guided wizard, AI-powered copy, auto-import from
     an existing website URL. First site live in under 2 hours.

  2. Control what goes live — a full approval workflow where admins see
     exactly what changed (old vs. new, field by field) before approving.

  3. Manage at scale — all sites, groups, and multi-location networks in
     one dashboard. 5 sites or 500 sites, same interface.

2. THE PROBLEM
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Multi-location businesses are stuck between two bad options:

  ✗ DIY builders (Wix, Squarespace) — no team controls, no approval
    process, any local manager can publish anything, instantly.

  ✗ Enterprise CMS (WordPress, Adobe AEM) — requires developers,
    costs hundreds of thousands of dollars, months to onboard.

The result: outdated websites, inconsistent branding, and IT teams
fielding "can you update my hours?" tickets from 30 locations.

Atlas eliminates this. Local staff fill in a form. Corporate approves.
Website goes live. No developers. No surprises.

4. HOW IT WORKS — THE FULL LIFECYCLE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  STEP 1 — SETUP (15–30 min)
  Local manager fills an 8-section guided wizard: business name,
  address, hours, services, team, colors, logo. Atlas builds the
  full website from those answers automatically.

  STEP 2 — DESIGN (30–60 min, optional)
  A 3-panel visual editor: section list on the left, live preview
  in the center, content controls on the right. Every change appears
  in the preview in real time. AI toolbar rewrites copy on demand.

  STEP 3 — SUBMIT
  Manager clicks "Submit for Review." Cannot publish directly.
  The system snapshots every single change made.

  STEP 4 — APPROVAL
  Admin sees a detailed diff — old value vs. new value for every
  changed field. Approve → live. Request Changes → written feedback
  sent back to the manager instantly.

  STEP 5 — LIVE & REPEAT
  Site is published. Any future update goes through Steps 2–4.
  Every version is stored permanently and can be rolled back.

5. CORE FEATURES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

DASHBOARD
  4 KPI cards (Total Sites, Published, Pending Approvals, Live Domains).
  Publication pipeline (Draft → Scheduled → Published → Live).
  Priority action list ("3 Approvals Due Today") with direct links.
  Real-time activity feed filterable by action type.

SETUP WIZARD (Step 1)
  8-section form covering business identity, service categories,
  location & contact, operating hours, services, team members,
  integrations, and footer. Only 4 required to proceed. Auto-saves
  continuously. Progress bar tracks completion in real time.

VISUAL EDITOR (Step 2)
  3-panel layout. 11 core page sections (Hero, Services, Team,
  Testimonials, FAQ, Contact, Footer, etc.) + 18 addable dynamic
  sections (Stats, Gallery, CTA Band, Feature Grid, etc.).
  Drag to reorder. Toggle visibility without deleting. Live preview
  at Desktop / Tablet / Mobile viewport sizes.

AI CONTENT TOOLS
  Every text field has a 5-action AI toolbar: Enhance, Rephrase,
  Shorten, Expand, Fix. One-click SEO generation writes the meta
  title, description, and focus keyword from business data. Tone
  presets rewrite all sections at once. "Fill from Scratch" wizard
  generates a complete site's content in one click.

SEO TAB
  Live SEO score (0–100) from 10 health checks. Google SERP and
  social OG card preview. All fields with character counters.

WEBSITE IMPORTER
  Enter any existing website URL → system extracts name, address,
  hours, colors, services, team, and more. Each field shows a
  confidence score. User picks what to import. Populates the entire
  setup form in under 2 minutes.

APPROVAL WORKFLOW
  Immutable version snapshots on every submission. Field-level diff
  engine shows exactly what changed. Admin approves, rejects, or
  requests changes with written feedback. Full version history.
  (This is the #1 differentiator — no other website builder has it.)

SITE MANAGEMENT
  All Sites — searchable table with status badges (Draft, Published,
  Scheduled, Live Domain).
  Groups — organize locations under a franchise or brand.
  Multi-Location — parent/child hierarchies for location networks.

SUPPORTING TOOLS
  Media Library — upload and reuse images and videos across sites.
  Banner Management — time-limited promos on any site, any placement.
  Data Collection — central catalogue of services, staff, and reviews.
  User Management — 3 roles (Admin, Manager, Editor), invite by email.
